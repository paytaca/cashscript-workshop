Defining a store

Open src/store/example-store.js

                    // src/store/example-store.js

                    import { defineStore } from "pinia";

                    export const useCounterStore = defineStore('counter', {
                        state: () => ({
                            counter: 0,
                        }),
                        getters: {
                            doubleCount: (state) => state.counter * 2,
                        },
                        actions: {
                            increment() {
                            this.counter++
                            },
                        },
                    })

Using the store

As a reactive state

                        
                        <script setup>
                        import { useCounterStore } from 'src/store/example-store.js';

                        const counterStore = useCounterStore();

                        function displayStore() {
                            console.log(counterStore.counter);
                            console.log(counterStore.doubleCount);
                        }
                        </script>

Using the store

Updating the store's state

                        
                        <script setup>
                        import { useCounterStore } from 'src/store/example-store.js';

                        const counterStore = useCounterStore();

                        function addFive() {
                            counterStore.counter = counterStore.counter + 5;
                        }
                        </script>

Creating quasar project

  1. Run command npm init quasar@latest
  2. Set configuration
    • Select: App with Quasar CLI
    • Set project folder name (e.g. "my-project")
    • Select script type: Javascript (or Typescript)
    • Select Quasar CLI with Webpack
    • Set package name (e.g. "my-project")
    • Set project name (e.g. "My Project")
    • Set project description
    • Select Vue component style: Options API
    • Select CSS preprocessor
    • Select features in project: Linting (ESLint), State Management (Pinia), axios
  3. Move to project folder cd my-project
  4. Install dependencies npm install
  5. Run server quasar dev

Group Work

https://bit.ly/cashscript-group-work

Fund Raiser

  • Multiple people can fund the contract
  • Funds will be released to the owner, when the goal amount is reached

Salary Distribution

  • Employer funds the contract with one transaction (or more)
  • Contract distributes the funds to multiple employees automatically according to their set amounts

Donor Splitter

  • Donors send to a single contract
  • When the goal amount is reached, funds are split to multiple charities that were initially set

Rock Paper Scissors Game

  • The owner O will set his/her choice (rock, paper, or scissors) , and fund the contract
  • During play, the contender C will pick a choice
  • If C wins, the funds goes to C
  • If O wins, the funds goes to O
  • If draw, the funds goes to O as well
