What is Proof-of-Work?

Proof-of-Work is a consensus mechanism where participants prove they've expended computational effort

  1. It prevents Sybil attack
  2. It determines which participant can append the next block of transactions
  3. It aligns the economic incentive to mine and the issuance of new Bitcoins

What is Sybil Attack?

Sybil attack is a type of attack where a participant creates multiple identities to gain control of the network

Consensus Rules Miners Must Follow

Miners must follow specific rules to create blocks that the network will accept

üîí Block Structure & Header Rules

  • Valid Merkle root

  • Timestamp within range

üí∞ Monetary & Reward Rules

  • Block reward ‚â§ subsidy + fees

  • 100 block maturity period

üßæ Transaction Validation Rules

  • No double-spending
  • Outputs ‚â§ inputs

‚õìÔ∏è Chain Selection Rule

Key Points About UTXOs

  • Each UTXO is a separate and can have any amount of funds (in satoshis)
  • An address can hold multiple UTXOs from different transactions
  • Total balance = Sum of all UTXOs in the address
  • To spend funds, you must reference specific UTXOs
