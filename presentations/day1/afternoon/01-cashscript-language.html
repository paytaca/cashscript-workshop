What is CashScript?

  • Smart contract language for Bitcoin Cash (2019)
  • Stateless programming
  • Based on Bitcoin Script
  • Compiles to bytecode

1. Contract Declaration

contract TransferWithTimeout(
    bytes20 sender,
    bytes20 recipient,
    int timeout
) {
    function transfer(sig s, pubkey pk) {
        require(checkSig(s, pk));
        require(hash160(pk) == sender);
        require(tx.time >= timeout);
        
        bytes34 senderOutput = new OutputP2PKH(recipient, tx.value);
    }
}
  • contract - keyword to define a smart contract
  • TransferWithTimeout - contract name
  • Parameters - contract inputs (sender, recipient, timeout)

                    }
                }
  • OutputP2PKH - creates a Pay-to-Public-Key-Hash output
  • recipient - destination address
  • tx.value - amount to transfer
  • This is the actual transfer action
