Cashscript Language

Day 1 - Afternoon Session

Writing Bitcoin Cash smart contracts

What is Cashscript?

  • Smart contract language for Bitcoin Cash
  • Based on Bitcoin Script
  • Compiles to bytecode
  • Type-safe and developer-friendly

Basic Syntax - Contract Structure

Let's analyze a complete contract step by step:

Complete Contract Overview

contract TransferWithTimeout(
    bytes20 sender,
    bytes20 recipient,
    int timeout
) {
    function transfer(sig s, pubkey pk) {
        require(checkSig(s, pk));
        require(hash160(pk) == sender);
        require(tx.time >= timeout);
        
        bytes34 senderOutput = new OutputP2PKH(recipient, tx.value);
    }
}

This contract allows time-locked transfers with signature verification!
