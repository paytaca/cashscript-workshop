Types & Operators

How To Do Operations?

Boolean

true || false
  • Operators:  !, &&, ||, ==, !=

Integer

1000000
1_000_000
1e6
1E6
  • Arithmetic Operators:   +, -, *, /, %
  • Comparison Operators:  <, >, >=, <=
  • Range:  -2^63 to 2^63 - 1 (Uses BigInt)

Date & Time

int timestamp = date("2021-02-17T01:30:00");
require(timestamp == 1613554200);
Dates are treated as integer on contracts and are always represented by date("YYYY-MM-DDThh:mm:ss")

String

"Bitcoin Cash"
'Bitcoin Cash'
  • Operators:   +
  • Comparison Operators:  !=, ==
  • Members:
    • length
    • split(int)
    • slice(int, int)
    • reverse()

Byte

0x123456
  • Operators:   +, &, |, ^
  • Comparison Operators:  !=, ==
  • Members:
    • length
    • split(int)
    • slice(int, int)
    • reverse()

Special Bytes Types

  • pubkey - Public Key (33 bytes)
  • sig - Transaction Signature (65 bytes)
  • datasig - Data Signature (64 bytes)
  • bytes20 - pubkey hashes
  • bytes32 - token categories
  • bytes35 - locking bytecodes
Operators used by these types are  ==, !=

Array

checkMultisig([sig1, sig2], [pk1, pk2, pk3]);
Arrays are not yet supported except for this function above

Tuple

string hello, string world = "Hello World".split(6);
Tuples are the type that is returned when calling the split member function on a string or bytes type
