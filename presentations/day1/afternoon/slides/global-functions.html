Global Functions

CashScript has several built-in functions for things like cryptographic and arithmetic applications,
It includes many common arithmetic and other operators that you would expect in a programming language.

Global Functions

Arithmetic Functions

        // Returns the absolute value of argument a.
        int abs(int a)
        

        // Returns the minimum value of arguments a and b.
        int min(int a, int b)
        

        // Returns the maximum value of arguments a and b.
        int max(int a, int b)
        

        // Returns true if and only if x >= low && x < high.
        int within(int x, int low, int high)

Global Functions

Hashing Functions

        // Returns the RIPEMD-160 hash of argument x.
        bytes20 ripemd160(any x)
        

        // Returns the SHA-1 hash of argument x.
        bytes20 sha1(int x)
        

        // Returns the SHA-256 hash of argument x.
        bytes32 sha256(any x)
        

        // Returns the RIPEMD-160 hash of the SHA-256 hash of argument x.
        bytes32 hash160(int x)
        

        // Returns the double SHA-256 hash of argument x.
        bytes32 hash256(int x)

Signature Checking Functions
            
                // Returns the RIPEMD-160 hash of argument x. 
                bool checkSig(sig s, pubkey pk)
            
        
            
                // Performs a multi-signature check using a
                // list of transaction signatures and public keys.
                bool checkMultiSig(sig[] sigs, pubkey[] pks)
            
        
            
                // Checks that sig s is a valid signature 
                // for message msg and matches with public key pk.
                bool checkDataSig(datasig s, bytes msg, pubkey pk)
