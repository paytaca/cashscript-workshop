Cashscript IDE

  • The Cashscript IDE is an online, web-based editor specifically designed for writing, testing, and deploying CashScript smart contracts.

Why it exists?

  • Built-in Compiler & Simulator

CashScript IDE: Compiles your CashScript code into Bitcoin Script automatically and includes a simulator to test contracts without using real BCH. In VS Code, separate tools and manual setup would be required.

Preloaded Libraries & Templates

Includes ready-made templates for common contracts (multisig, timelock, password). No need to manually copy boilerplate code.

Beginner-Friendly

Everything is integrated in the browser â€” no Node.js setup or dependencies needed. Great for learning, experimenting, and teaching.

CashScript IDE Tabs

  1. Editor: Where you write your CashScript smart contract.
  2. New Contract: Instantiates a contract; behind the scenes, it initializes a Contract using the CashScript SDK.
  3. Contracts: Shows the list of contracts you've created in the IDE.
  4. Wallets: Interface for managing test wallets and keys. Required if your contract needs a signature.
  5. Transaction Builder: GUI for creating transactions using your contract or a wallet; easier than manually coding in JavaScript.
