What is CashScript?

CashScript

A high-level programming language used specifically to write Bitcoin Cash contracts.

CashScript

Created by Rosco Kalis (May 2019)

Stateless Program


// stateless programming = outputs depend only on the inputs
function add(a, b) {
  return a + b;
}

// stateful programming = outputs change each time you call it
let counter = 0;

function increment() {
  counter++;
  return counter;
}

Each execution depends only on the inputs, and the output will always be the same for the same arguments
